# blogging-cookbook

This cookbook installs a blogging platform, based on nginx, php, wordpress, pure-ftpd, mysql and phpmyadmin

## Supported Platforms

* Debian, Ubuntu
* CentOS, Red Hat

## Usage

### blogging::default

Include `blogging` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[blogging::default]"
  ]
}
```
